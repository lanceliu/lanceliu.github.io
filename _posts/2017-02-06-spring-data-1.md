# Spring Data

Spring Data`s 任务是提供一个熟悉一致的数据获取编程模型。它使得数据访问技术更加容易，关系和非关系数据库，map-reduce frameworks, and cloud-based data services。这是一个综合性的项目，包含很多子工程对于特定的数据库。
